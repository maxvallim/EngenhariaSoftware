Cheatsheet — PlantUML

Principais comandos para Diagramas de Atividade

Propósito deste Cheatsheet

Fornecer um guia rápido com os comandos essenciais do PlantUML para criar Diagramas de Atividade, seguindo o padrão utilizado em sala de aula.

1) Estrutura básica

@startuml
... elementos ...
@enduml

2) Ação (Activity)

:Descrição da ação;
:Validar credenciais;
:Atualizar estoque;

3) Fluxo sequencial

@startuml
start
:Ação 1;
:Ação 2;
:Ação 3;
stop
@enduml

4) Decisão (if/else)

if (Condição?) then (sim)
  :Ação se verdadeiro;
else (não)
  :Ação se falso;
endif

5) Fork/Join (Ações paralelas)

fork
  :Ação paralela 1;
fork again
  :Ação paralela 2;
fork again
  :Ação paralela 3;
end fork

6) Swimlanes (faixas de responsabilidade)

partition Usuario {
  :Informa dados;
}

partition Sistema {
  :Valida credenciais;
}

7) Swimlanes com Pipe (|)

@startuml
|Usuário|
start
:Informa credenciais;

|Sistema|
:Valida credenciais;

|Usuário|
if (Válidas?) then (sim)
  :Acessa dashboard;
else (não)
  :Exibe erro;
endif

stop
@enduml

8) Exemplo completo e funcional

@startuml
top to bottom direction

start
:Iniciar pedido;

if (Pagamento aprovado?) then (sim)
  fork
    :Enviar confirmação;
  fork again
    :Atualizar estoque;
  end fork
else (não)
  :Exibir erro;
endif

:Finalizar;
stop
@enduml
