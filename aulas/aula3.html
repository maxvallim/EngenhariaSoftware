Aula 03 — Requisitos Funcionais (RF), Requisitos Não Funcionais (RNF) e Regras de Negócio (RN)

Objetivos da aula

  • Compreender a diferença entre RF, RNF e RN.
  • Escrever requisitos claros, objetivos e verificáveis.
  • Usar a estrutura mínima recomendada para cada tipo de requisito.

1) Conceitos Essenciais

Requisito Funcional (RF)

Define o que o sistema deve fazer — funcionalidades, ações e comportamentos observáveis.

RF01 — Login do usuário
O sistema deve permitir login com e-mail e senha.

Um RF sempre descreve um comportamento concreto.

Requisito Não Funcional (RNF)

Define como o sistema deve se comportar — desempenho, segurança, disponibilidade, qualidade.

RNF01 — Performance de login
O sistema deve responder à tentativa de login em até 2 segundos.

Um RNF SEMPRE precisa de métrica.

Regra de Negócio (RN)

Regras, leis, políticas e restrições que pertencem ao domínio e influenciam a lógica do sistema.

RN01 — Bloqueio de login
Após 5 tentativas inválidas, bloquear por 15 minutos.

Uma RN deve ser determinística (verdadeira ou falsa).

Exemplos (opcional).

3) Exemplos

Requisitos Funcionais (RF)

RF01 — Login
Permitir autenticação via e-mail e senha.

RF02 — Recuperação de senha
Enviar link de redefinição por e-mail.

RF03 — Atualização de perfil
Permitir alteração de nome, foto e dados pessoais.

Requisitos Não Funcionais (RNF)

RNF01 — Tempo de resposta
O login deve responder em até 2 segundos.

RNF02 — Disponibilidade
O sistema deve estar disponível 99,5% do tempo mensal.

RNF03 — Segurança
Senhas devem ser armazenadas com hashing seguro.

Regras de Negócio (RN)

RN01 — Bloqueio após tentativas
Após 5 erros, bloquear o login por 15 minutos.

RN02 — Maioridade
Para cadastro, o usuário deve ter 18 anos ou mais.

RN03 — Cancelamento
Pedidos só podem ser cancelados antes do envio.

6) Dicas finais

  • Use verbos claros: permitir, enviar, registrar, calcular, validar.
  • Evite termos vagos: “rápido”, “seguro”, “bonito”.
  • RNF SEMPRE precisa ser mensurável.
  • RN deve ser determinística (não ambígua).
  • Padronize nomenclatura: RF01, RNF01, RN01…
