Cheatsheet — PlantUML

Principais comandos para Diagramas de Caso de Uso

Propósito deste Cheatsheet

Listar de forma clara e rápida os comandos mais usados em PlantUML para criar diagramas de caso de uso, exatamente como aplicamos nas aulas.

1) Estrutura básica do arquivo

@startuml
... elementos ...
@enduml

2) Ator

actor Usuario
actor "Cliente Premium" as CP
actor "Sistema Externo" as Ext

3) Caso de Uso

(Login)
(Realizar Pedido)
(Consultar Histórico) as Hist
("Cadastrar\nNovo Usuário") as CadNovo

4) Relacionamentos principais

Associação (ator → caso de uso)

Usuario --> (Login)
Cliente --> (Realizar Pedido)

Include

(Realizar Pedido) --> (Calcular Frete) : <>

Extend

(Realizar Pedido) --> (Aplicar Cupom) : <>

Generalização (herança de atores)

actor Cliente
actor "Cliente Premium" as CP

CP --|> Cliente

5) Pacotes e organização visual

package "Pagamentos" {
  (Efetuar Pagamento)
  (Validar Cartão)
}

rectangle Sistema {
  (Login)
  (Cadastro)
}

6) Direção do layout

left to right direction
top to bottom direction

7) Notas

note right of (Login)
  Fluxo principal: usuário informa e-mail e senha.
end note

(Login) .. note right : Fluxo alternativo A1

8) Estereótipos (ótimo para MVP, categorias etc.)

(Login) <>
(Consultar Histórico) <>

skinparam usecaseBackgroundColor<> #C6F6D5
skinparam usecaseBorderColor<> #2F855A

9) Exemplo completo e funcional

@startuml
left to right direction
actor Cliente
actor Admin

rectangle Sistema {
  (Login)
  (Cadastro)
  (Realizar Pedido)
  (Consultar Histórico)
}

Cliente --> (Login)
Cliente --> (Realizar Pedido)
Cliente --> (Consultar Histórico)
Admin   --> (Cadastro)

(Realizar Pedido) --> (Calcular Frete) : <>
(Realizar Pedido) --> (Aplicar Cupom)  : <>
@enduml
